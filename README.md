# Крестики-нолики(XandO)
Это игра в крестики-нолики была создана на python(pygame), он также необходим для запуска

- **Всегда** сначала ходит крестик(первый игрок), а потом нолик(второй игрок соответственно), ход произходит при нажатии левой кнопки мыши
- При выкладывании игроком 3-х фигур в ряд он выигрывает
- Если никто к концу игры не выстроил ряд из фигур, игра заканчивается ничьёй
